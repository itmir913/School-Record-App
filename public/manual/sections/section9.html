9. 유의어 점검 (Inspect)

좌측 메뉴에서 유의어 점검(Inspect) 탭을 선택합니다. NEIS 제출 전 생기부에서 쓰지 말아야 할 어휘(제한 표현·유의어)를 자동으로 찾아 강조 표시합니다. 총 4단계 마법사 형식으로 진행됩니다.

9-1. Step 1 — 유의어 관리

점검에 사용할 유의어를 그룹 단위로 관리합니다. 그룹을 만들고, 그룹 안에 검색할 단어를 등록합니다.

실제 화면 — Step 1. 유의어 관리

유의어 점검 (Inspect)

중·고등학교 학교생활기록부 기재요령에 근거하여 유의어 및 금지어를 점검합니다.

기능 설명
그룹 추가 화면 하단 + 그룹 추가 버튼으로 새 그룹을 만듭니다. 그룹은 점검 목적별로 나눌 수 있습니다(예: 유의어_기본, 학교 자체 금지어).
단어 추가 그룹 카드 하단 입력창에 단어를 입력하고 추가 버튼(또는 Enter)을 누릅니다. 이미 등록된 단어는 칩(chip) 형태로 표시됩니다.
단어 삭제 칩 오른쪽 ✕ 버튼을 클릭하면 해당 단어가 삭제됩니다.
유의어 일괄 등록 그룹 카드 하단 유의어 일괄 등록을 펼치면 여러 단어를 줄바꿈 또는 쉼표로 구분하여 한 번에 등록할 수 있습니다.
그룹 삭제 그룹 헤더 오른쪽 🗑 버튼을 클릭하면 그룹과 포함된 단어가 모두 삭제됩니다. 삭제 전 확인 창이 표시됩니다.
💡

유의어 목록은 DB 파일에 저장됩니다. 한 번 등록해두면 이후 실행 시에도 유지됩니다. 단어 관리가 완료되면 하단 다음 → 버튼으로 Step 2로 이동합니다.

9-2. Step 2 — 점검할 그룹 선택

Step 1에서 만든 유의어 그룹 중, 이번 점검에 사용할 그룹을 하나 이상 선택합니다.

실제 화면 — Step 2. 점검할 그룹 선택

✓

6개 단어

학교 자체 금지어

3개 단어

선택된 그룹: 1개  |  중복 제거 단어: 6개

카드를 클릭하면 선택/해제됩니다. 여러 그룹을 동시에 선택하면 단어가 합쳐져 점검됩니다(중복 단어는 한 번만 검색). 그룹을 하나 이상 선택해야 다음 → 버튼이 활성화됩니다.

9-3. Step 3 — 점검 범위 선택

유의어를 점검할 생기부 범위를 선택한 후 검색 시작 버튼을 누릅니다.

실제 화면 — Step 3. 점검 범위 선택

✓

전체 영역

특정 영역

점검할 영역을 직접 선택합니다

🔍 검색 시작
범위 옵션 설명
전체 영역 DB에 저장된 모든 학생의 모든 영역·활동 기록을 점검합니다. 제출 직전 최종 점검에 적합합니다.
특정 영역 선택한 영역만 점검합니다. 특정 영역을 선택하면 영역 카드 목록이 활성화되며, 하나 이상의 영역을 클릭하여 선택할 수 있습니다.

9-4. Step 4 — 점검 결과

검색이 완료되면 유의어가 탐지된 기록 목록이 테이블로 표시됩니다. 결과가 있으면 우측 상단에 Excel 내보내기 버튼이 활성화됩니다.

9-5. Excel 내보내기

Step 4 결과 화면 우측 상단의 Excel 내보내기 버튼을 클릭하면 점검 결과를 엑셀 파일로 저장할 수 있습니다. 탐지된 기록이 한 건도 없으면 버튼이 비활성화됩니다.

1 Excel 내보내기 버튼을 클릭하면 파일 저장 대화상자가 열립니다.
2 저장 위치를 선택하고 확인하면 유의어점검결과_YYYYMMDD_HHmm.xlsx 형태로 저장됩니다.
3 저장 완료 화면에서 파일 확인 버튼을 클릭하면 파일 탐색기에서 해당 파일 위치가 열립니다.

9-6. 기본 제공 유의어 목록

이 프로그램에는 개발자가 학교생활기록부 기재요령 및 교육부·시도교육청 안내 자료를 바탕으로 미리 작성한 유의어 목록이 기본 내장되어 있습니다. 프로그램을 처음 실행하면 이 목록이 자동으로 등록되며, Step 1에서 언제든지 추가·수정·삭제할 수 있습니다.

기본 목록은 기재요령상 기재 금지·제한 표현, 부모 배경·경제력 노출로 이어질 수 있는 어휘를 대상으로 합니다. 아래는 그 예시입니다.

점검 표현 처리 방향
논문 → "탐구 보고서", "소논문 형식의 보고서"로 대체
연구 → "탐구", "조사", "분석" 등으로 대체
모의고사 → 해당 내용 삭제. 교내 수업·활동 중심으로 재기술
출판 → "제작", "작성", "발표" 등으로 대체하거나 삭제
부모님 → 보호자 관련 내용 전체 삭제
해외 → 해당 활동 내용 삭제 또는 교내·국내 활동으로 재기술
자격증 → 기재 가능 여부 확인 후, 불가 시 관련 학습 활동 내용으로 대체
⚠️

기본 목록은 일반적인 기재요령을 기준으로 작성된 것으로, 학교·학년·담당자마다 세부 지침이 다를 수 있습니다. 학교 생기부 담당자의 안내를 우선 따르고, 필요에 따라 Step 1에서 목록을 직접 수정하세요.
