8. 텍스트 치환 (Replace)

좌측 메뉴에서 텍스트 치환(Replace) 탭을 선택합니다. 자주 반복되는 표현을 일괄 교체하거나, 불필요한 공백·줄바꿈을 정리합니다.

8-1. 기본 제공 치환 규칙

프로그램에는 몇 가지 기본 치환 규칙이 내장되어 있습니다.

실제 화면 — 텍스트 치환(Replace) 규칙 목록

텍스트 치환 (Replace)

생기부 내용에서 텍스트를 찾아 일괄 교체합니다.

전체 적용 규칙 추가
기본
정규식 \n+ → (공백)
연속된 줄바꿈 제거
기본
정규식 {2,} → (단일 공백)
연속 공백 단일화
사용자
보여줌 → 보임
어색한 표현 교정
규칙 찾을 내용 바꿀 내용 목적
기본 1 \n+ 공백 1개 줄바꿈 → 공백으로 정리
기본 2   (공백 2개 이상) 공백 1개 연속 공백 단일화

8-1-1. 사용자 정의 규칙 예시

생기부에서 자주 발생하는 어색한 표현을 교정하는 규칙 예시입니다. 아래 표현들을 규칙으로 등록해두면 전체 적용 한 번으로 전 학생 기록을 일괄 정리할 수 있습니다.

찾을 내용 바꿀 내용 설명
보여줌 보임 구어체 → 서술형 교정
하였음 함 간결한 서술형으로 통일
~이다 ~임 생기부 어미 형식 통일
기여하였다 기여함 어미 통일
있었음 있음 과거 시제 중복 제거
OO 학생 (해당 이름) 가져오기한 파일에 "OO 학생" 표기가 일괄 포함된 경우

8-2. 사용자 정의 규칙 추가

  1. 1 규칙 추가 클릭 → 찾을 내용과 바꿀 내용을 입력하고 저장합니다.
  2. 2 정규식(Regex) 토글을 켜면 고급 패턴을 사용할 수 있습니다.
  3. 3 규칙 목록을 확인한 후 전체 적용 버튼을 클릭하면 모든 영역·활동의 기록에 일괄 적용됩니다.
⚠️

전체 적용은 되돌릴 수 없습니다

전체 적용 전에 스냅샷을 생성하거나, 적용 전 미리보기를 통해 바뀔 내용을 먼저 확인하세요. 정규식 패턴이 잘못되면 의도치 않은 내용이 삭제될 수 있습니다.
