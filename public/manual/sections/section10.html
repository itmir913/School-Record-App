10. 내보내기 (Export)

좌측 메뉴에서 내보내기(Export) 탭을 선택합니다. 완성된 학교생활기록부를 다양한 형식의 엑셀 파일(.xlsx)로 저장합니다.

10-1. 내보내기 절차

실제 화면 — 내보내기(Export)

데이터 내보내기(Export)

완성된 학교생활기록부를 다양한 형식으로 내보냅니다.

ℹ 내보내기 안내

이 파일은 학교생활기록부 작성을 돕기 위한 참고용 자료입니다.
반드시 담당 선생님께서 내용을 직접 검토하신 후 나이스(NEIS)에 입력해 주시기를 부탁드립니다.
내보내기를 실행하면 위 안내 사항을 확인하신 것으로 간주합니다.

1

내보내기 형식 선택

A·B·C 세 가지 엑셀 형식 중 하나를 선택합니다. 각 형식의 열 구성은 10-2를 참고하세요.

2

영역 선택

내보낼 영역을 하나 선택합니다. 선택한 영역에 등록된 모든 학생과 활동이 파일에 포함됩니다.

3

내보내기 실행

유형·영역·학생 수·활동 수·예상 행 수를 확인한 뒤 내보내기 실행 버튼을 클릭합니다. 저장 경로 대화상자가 열리면 파일 이름과 위치를 지정하면 .xlsx 파일이 생성됩니다.

✓

내보내기 완료

저장된 행 수와 파일 이름이 표시됩니다. 파일 확인 버튼으로 탐색기에서 파일을 바로 열거나, 새로 내보내기로 처음부터 다시 시작할 수 있습니다.

10-2. 출력 파일 형식

A 타입 — 행 단위 활동 형식

학생 1명의 활동 1개가 한 행에 기록됩니다. 학생 1명 = 여러 행.

열 내용
A학년
B반
C번호
D이름
E활동명
F활동내용

B 타입 — 열 단위 활동 형식

활동이 열(헤더)로 구분됩니다. 학생 1명의 모든 활동이 한 행에 기록됩니다. 학생 1명 = 1행.

열 내용
A학년
B반
C번호
D이름
E…활동명 1, 활동명 2, … (활동별 열)

C 타입 — 최종 나이스(NEIS) 문장 형식 (추천)

모든 활동 내용을 공백으로 연결한 최종 문장 1개가 한 열에 들어갑니다. NEIS 직접 입력용 최종 문장 생성에 적합합니다.

열 내용
A학년
B반
C번호
D이름
E영역명 — 활동 순서대로 연결된 최종 문장
