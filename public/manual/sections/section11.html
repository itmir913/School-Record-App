11. 부록

11-1. 체크리스트 내보내기

좌측 메뉴에서 체크리스트(Checklist) 탭을 선택합니다. 어떤 학생이 어떤 활동에 참여했는지 한눈에 볼 수 있는 체크리스트 엑셀 파일을 생성합니다. 단면 인쇄하여 최종 점검 때 활용하면 누락 없이 생기부를 마무리하는 데 도움이 됩니다.

  1. 1 체크리스트에 포함할 영역을 선택합니다.
  2. 2 엑셀 내보내기 버튼을 클릭하면 학생별·활동별 작성 완료 여부가 담긴 파일이 저장됩니다.

11-2. 설정 (Settings)

좌측 메뉴 하단의 설정(Settings) 탭에서 프로그램 동작을 조정할 수 있습니다.

  • 테마 설정: 다크 모드와 라이트 모드를 전환합니다. 생기부 작성 탭 우측 상단 버튼에서도 동일하게 변경할 수 있습니다.
  • 암호화 설정: DB 파일 암호화를 활성화하거나 비밀번호를 변경합니다. 변경 시 즉시 적용됩니다.

11-3. 데이터 백업 및 복구

이 프로그램은 몇 가지 안전장치로 데이터를 보호합니다.

💾

입력 즉시 자동 저장

글자를 입력하는 순간 DB에 저장됩니다. 별도 저장 버튼이 없으며, 예기치 않은 종료 시에도 데이터 유실이 없습니다.

📋

셀 히스토리

셀마다 변경 이력이 누적되어 이전 버전으로 돌아갈 수 있습니다.

📷

스냅샷

원하는 시점에 DB 전체를 저장하여 언제든 해당 시점으로 복원합니다.

🔄

자동 이중 백업

프로그램을 열 때마다 DB 파일의 복사본을 자동으로 생성합니다. DB 파일과 동일한 위치에 .db.backup 파일이 생성됩니다.

DB 파일 손상 시 복구 방법

  1. 프로그램을 완전히 종료합니다.
  2. 손상된 .db 파일을 삭제하거나 다른 이름으로 변경합니다.
  3. .db.backup 파일의 이름에서 .backup 부분을 제거합니다.
  4. 프로그램을 다시 실행하여 복구된 파일을 엽니다.

11-4. 자주 묻는 질문 (FAQ)

Q. 프로그램을 다른 PC로 이동하려면 어떻게 하나요?

.db 파일만 복사하면 됩니다. 새 PC에 프로그램을 설치한 뒤 해당 파일을 열면 모든 데이터가 그대로 이어집니다.

Q. 학생 데이터가 외부로 전송되지 않나요?

이 프로그램은 완전 오프라인으로 동작합니다. 모든 데이터는 선생님의 PC에 있는 .db 파일에만 저장되며, 어떠한 서버로도 전송되지 않습니다.

Q. 영역에 배정된 학생을 나중에 추가하거나 제외할 수 있나요?

네. 영역(Area) 탭에서 해당 영역의 학생 배정 버튼을 다시 클릭하면 배정된 학생을 변경할 수 있습니다. 이미 입력된 기록은 배정 변경과 무관하게 유지됩니다.

Q. 암호화 비밀번호를 잊어버렸습니다.

암호화된 DB 파일은 비밀번호 없이는 복구할 수 없습니다. .db.backup 파일이 있다면 해당 파일의 암호화 설정을 확인하세요. 암호화를 사용하는 경우 비밀번호는 반드시 안전한 곳에 별도 보관하시기 바랍니다.

Q. NEIS에 업로드할 때 오류가 발생합니다.

바이트 초과 여부를 먼저 확인하세요. 생기부 작성 탭에서 붉은색으로 표시된 학생이 없는지 점검한 뒤 다시 내보내기를 시도합니다. 그래도 오류가 발생한다면 NEIS 안내 문서 또는 교육청 담당자에게 문의하세요.
