2. 기본 개념 이해

2-1. 영역(Area)과 활동(Activity) 구조

이 프로그램은 학교생활기록부를 영역(Area)과 활동(Activity)의 2단계 구조로 관리합니다.

영역 (Area) = 생기부 항목 하나 (예: 진로활동, 자율활동, 교과세특 등)
활동 (Activity) = 영역 안의 세부 기록 단위 (예: 체험학습, 수행평가1 등)
활동 (Activity) = 활동은 영역 하나에 여러 개 추가할 수 있습니다.
활동 (Activity) …
각 활동별로 학생마다 내용을 입력하면, 프로그램이 모든 활동의 기록을 하나로 합산하여 해당 영역의 최종 문장을 완성합니다.
역할 영역(Area) 예시 활동(Activity) 예시
담임교사 1반 자율활동 학급자치회, 스포츠데이, 수학여행
담임교사 1반 진로활동 전공탐색 퀴즈, 진로 상담, 진로 독후감
동아리교사 컴퓨터 동아리 동아리 실험, 동아리 프로젝트, 동아리 발표회
교과교사 프로그래밍 A반 수행평가1, 수행평가2, 수업태도
교과교사 프로그래밍 B반 수행평가1, 수행평가2, 수업태도

1

학생 등록 → 학생(Students)

담당 학생의 학년·반·번호·이름을 등록합니다. 출석부 엑셀 파일로 일괄 추가하면 빠릅니다.

2

영역 구성 → 영역(Area)

NEIS에 입력할 생기부 항목(자율활동, 진로활동, 교과세특 등)마다 영역을 만들고, 바이트 한도와 배정 학생을 설정합니다.

3

활동 등록 → 활동(Activity)

각 영역 안에 세부 활동(체험학습, 수행평가1, 독후감 등)을 추가합니다. 활동을 여러 개 만들수록 나중에 세밀하게 내용을 나눠 입력할 수 있습니다.

4

내용 작성 → 생기부 작성(Write) / 가져오기(Import)

활동별로 각 학생의 내용을 직접 입력하거나, 다른 교사가 작성한 엑셀 파일을 불러와 병합합니다.

5

검수 → 텍스트 치환(Replace) / 유의어 점검(Inspect)

반복 표현을 일괄 교체하고, NEIS에서 사용이 제한된 어휘를 자동 탐지합니다.

6

내보내기 → 내보내기(Export)

모든 활동의 내용을 합산하여 NEIS 업로드용 엑셀 파일로 저장합니다.

2-3. 역할별 활용 예시

담임교사

자율활동 · 동아리활동 · 진로활동 영역을 생성하고, 각 활동(체육대회, 체험학습 등)을 등록합니다. 지도교사들이 작성한 엑셀 파일을 가져오기(Import)로 불러와 병합·검수합니다. 바이트 초과 셀은 붉은색으로 강조되어 분량 조절이 쉽습니다.

교과교사

담당 과목별로 영역(분반)을 만들고, 수행평가·수업 참여 등을 활동으로 등록합니다. 엑셀과 유사한 격자 화면에서 여러 학급 학생의 세특을 화면 전환 없이 연속으로 입력합니다. 완성 후 내보내기(Export)로 담임교사에게 전달할 엑셀 파일을 생성합니다.
