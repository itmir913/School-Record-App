3-1. 학생 개별 추가

  1. 1 화면 오른쪽 위의 학생 추가 버튼을 클릭합니다.
  2. 2 팝업 창에서 학년, 반, 번호, 이름을 입력하고 저장합니다.

3-2. 엑셀 일괄 추가

기존에 사용하던 출석부(명렬표) 엑셀 파일을 그대로 불러올 수 있습니다. Excel 프로그램을 권장하며, 한셀의 경우 CSV 파일로 변환 후 불러오기를 권장합니다.

  1. 1 화면 오른쪽 위의 일괄 추가 버튼을 클릭합니다.
  2. 2 파일 선택 창에서 .xlsx 또는 .csv 파일을 선택하거나 화면에 끌어다 놓습니다.
  3. 3 열 이름이 아래 표의 표준 형식이면 자동으로 매핑됩니다. 그렇지 않으면 드롭다운으로 직접 지정합니다.
  4. 4 미리보기로 데이터를 확인한 뒤 가져오기를 클릭합니다.

예시 — 출석부 엑셀 파일 형태 (이런 파일을 그대로 사용할 수 있습니다)

학년 반 번호 이름
111학생A
112학생B
113학생C
121학생D
122학생E

열 이름이 아래 표에 해당하면 자동 인식됩니다. 열 이름이 다를 경우 드롭다운으로 직접 지정합니다.

3-3. 학생 수정 및 삭제

학생 행 오른쪽의 아이콘을 클릭하면 수정 팝업이 열립니다. 삭제 시 해당 학생의 모든 생기부 기록도 함께 삭제되므로 주의하세요.

3-4. 영역(Area)에 학생 일괄 배정하기

0.2.14 버전부터 각 영역마다 학생을 일괄 배정할 수 있는 기능이 추가되었습니다.

영역(Area) 관리 탭의 카드 요소 중 학생 배정 버튼을 누르면 뜨는 팝업 창에서 엑셀로 일괄 배정을 클릭하세요. 엑셀 명렬 파일을 가져와 해당 학생들을 영역에 바로 배정할 수 있습니다.

즉, 학생(Students) 관리 탭에서 학생을 미리 추가하지 않아도, 선택교과 A·B·C반처럼 영역별로 다른 수강생 명렬표를 각각 불러와 일괄 등록할 수 있습니다.

활용 예시 — 나이스에서 교과별 출석부 엑셀 파일을 다운로드한 뒤, 본 프로그램의 양식에 붙여넣으면 여러 반이 섞인 선택교과별 학생 명단을 손쉽게 불러올 수 있습니다.
