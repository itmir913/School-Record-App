# All-in-One 학교생활기록부 에디터

> **Teacher Utility Kit** — 교사를 위한 학교생활기록부 작성 도구

학생 명렬표 등록부터 영역 구성, 활동 생성, 생기부 작성, 내보내기까지.  
아래 순서대로 진행하면 학생별 학교생활기록부 문장을 손쉽게 완성할 수 있는 데스크톱 애플리케이션입니다.

---

## 주요 기능

- **학생 관리** — 학년·반·번호·이름 등록, 엑셀 일괄 가져오기 지원
- **영역 설정** — 자율활동, 진로활동, 동아리활동 등 영역별 NEIS 기준 바이트 제한 설정
- **활동 관리** — 세부 활동(수행평가, 프로젝트 등)을 영역과 연결하여 체계적으로 관리
- **특기사항 입력** — 학생 × 활동 격자 형태의 직관적인 입력 화면, 실시간 바이트 집계
- **유의어 점검** — 작성 단계에서 스마트한 유의어 오류 실시간 점검
- **텍스트 일괄 치환** — 정규식 지원, 전체 학생 기록을 클릭 몇 번으로 일괄 수정
- **누락 방지 체크리스트** — 필수 활동 누락 여부 확인용 엑셀 다운로드
- **저장 & 복구** — 입력 즉시 자동 저장, 버전 히스토리, 타임머신 스냅샷
- **엑셀 가져오기/내보내기** — 나이스 일괄 업로드 양식으로 내보내기, 기존 파일 가져오기 지원
- **자동 업데이트 확인** — GitHub 릴리즈 기반 신규 버전 알림

## 데이터 구조

생기부의 각 항목을 **영역(Area)** 이라고 부릅니다. 그 안에 학생이 실제로 참여한 개별 **활동(Activity)** 여러 개를 담아 하나의 문장 묶음을 완성합니다.

```
영역 (Area)  ───  진로활동  [최대 1,500 byte]
  │
  ├─ 활동 (Activity)  전공탐색 퀴즈
  ├─ 활동 (Activity)  전공 도서 독후감
  ├─ 활동 (Activity)  진로 상담 세션
  └─ 활동 (Activity)  ...
```

각 Activity마다 학생별로 기록을 작성하면, 프로그램이 합산된 문장을 해당 영역(Area) 항목으로 자동 병합합니다.

## 5단계 워크플로

| 단계 | 작업 | 설명 |
|:---:|---|---|
| 1 | **학생 등록** | 학년·반·번호·이름 등록. 엑셀(.xlsx, .csv)로 전체 명렬표 일괄 가져오기 가능 |
| 2 | **영역 구성** | 자율활동·진로활동·동아리활동 등 생기부 영역 생성 및 NEIS 기준 바이트 제한 설정 |
| 3 | **활동 생성** | 영역 하위에 세부 활동(수행평가, 프로젝트, 체험학습 등) 생성 |
| 4 | **생기부 작성** | 학생 × 활동 격자 화면에서 내용 입력. 실시간 합산 바이트 초과 시 셀 붉은색 강조 |
| 5 | **내보내기** | 나이스 일괄 업로드 양식에 맞춘 엑셀 파일로 내보내기 |

## 대상별 활용 가이드

### 담임교사 — 활동 취합 및 총괄 관리

자율활동·동아리활동·진로활동 영역을 병합하고 검수해야 하는 담임교사에게 최적화되어 있습니다.  
지도교사들이 작성한 엑셀 데이터를 **가져오기(Import)** 하여, 초과된 바이트를 한눈에 파악하고 효율적으로 분량을 조절할 수 있습니다.

> **활용 팁:** 바이트 초과 시 해당 셀이 붉은색으로 즉시 강조되므로, 길이를 줄여야 하는 학생을 직관적으로 찾아낼 수 있습니다.

### 교과교사 — 세특 작성

세부능력 및 특기사항(세특)을 집중적으로 작성하는 교과교사에게 최적화되어 있습니다.  
수행평가, 수업 참여도, 프로젝트 등 평가 요소들을 세부 활동(Activity)으로 만들어 관리하세요.

> **활용 팁:** 엑셀 스프레드시트와 유사한 격자형 인터페이스를 제공하여, 여러 학급과 학생의 세특을 화면 전환 없이 한 번에 연속해서 작성하기 편리합니다.

## 저장 & 복구

- **입력 즉시 자동 저장** — 글자를 타이핑하는 매 순간 데이터베이스에 즉시 저장. 예기치 못한 종료가 발생해도 데이터 유실 걱정 없음.
- **버전 히스토리** — 작성한 문구가 마음에 들면 지금 버전을 명시적으로 저장. 과거부터 누적된 모든 변경 사항을 추적.
- **타임머신 스냅샷** — 현재 생기부 전체를 스냅샷으로 저장. 대대적인 수정 후에도 원하는 시점으로 언제든지 되돌아올 수 있음.

## 개발 환경 설정

### 사전 준비

- [Node.js](https://nodejs.org/) (version 18 이상)
- [Rust 툴체인](https://rustup.rs/)

### 실행

```bash
# 의존성 설치
npm install

# 개발 모드 실행 (Vite + Tauri)
npm run tauri dev
```

### 테스트

```bash
# 전체 단위 테스트
npm run test
```

### 빌드

```bash
# 전체 빌드
npm run tauri build

# 포터블 실행 파일 빌드
npm run tauri:portable

# 설치 파일 (NSIS) 빌드
npm run tauri:installer
```
