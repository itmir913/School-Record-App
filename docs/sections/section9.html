9. 유의어 점검 (Inspect)

좌측 메뉴에서 유의어 점검(Inspect) 탭을 선택합니다. 학교생활기록부에 사용이 제한된 어휘를 자동으로 탐지하고, 적절한 대체 표현을 함께 제시합니다. 제출 전 최종 검수 단계에서 활용하세요.

동작 원리

이 프로그램은 교육부 및 시·도 교육청 기준의 생기부 유의어 목록을 내장하고 있습니다. DB에 저장된 모든 학생의 기록을 순회하며 유의어가 포함된 셀을 자동으로 탐지합니다.

  • 탐지 대상: 현재 DB에 저장된 모든 영역·학생·활동의 기록
  • 제시 정보: 발견된 유의어, 위치(영역·학생·활동), 권장 대체 표현
  • 강조 표시: 원문에서 유의어가 포함된 위치를 황색(amber)으로 하이라이트

사용 방법

  1. 1 유의어 점검 탭에서 점검 실행 버튼을 클릭합니다.
  2. 2 결과 목록에서 유의어가 발견된 셀마다 원문과 대체 표현이 나란히 표시됩니다. 원문의 유의어 위치는 황색으로 하이라이트됩니다.
  3. 3 결과를 참고하여 생기부 작성 탭에서 해당 내용을 직접 수정하거나, 텍스트 치환(Replace) 기능으로 일괄 교체합니다.

유의어 목록 관리

설정(Settings) 탭 또는 유의어 점검 탭 내의 유의어 관리 버튼에서 유의어 목록을 직접 추가·수정·삭제할 수 있습니다. 학교·지역별 추가 금지 어휘를 등록해 두면 더욱 정밀한 점검이 가능합니다.

💡

동일한 유의어가 여러 학생에게서 발견된다면, 치환(Replace) 기능을 이용해 전체 일괄 교체하는 것이 효율적입니다.
