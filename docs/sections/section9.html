9. 유의어 점검 (Inspect)

좌측 메뉴에서 유의어 점검(Inspect) 탭을 선택합니다. NEIS 제출 전 생기부에서 쓰지 말아야 할 어휘(제한 표현·유의어)를 자동으로 찾아 강조 표시합니다.

9-1. 화면 구성

실제 화면 — 유의어 점검(Inspect) 결과

유의어 점검 (Inspect)

작성된 내용에서 제한 표현을 찾아 강조합니다.

점검 실행
홍길동 1반 자율활동 › 학급자치회
학급 회의에서 적극적으로 의견을 제시하였으며 친구들의 이야기를 경청하고 뛰어난 리더십을 발휘함.
이순신 1반 진로활동 › 진로 상담
자신의 탁월한 분석력과 우수한 문제 해결 능력을 바탕으로 진로를 구체적으로 설계함.
강감찬 1반 자율활동 › 스포츠데이
유의어 없음
UI 요소 설명
뛰어난 호박색(amber) 하이라이트 = 유의어 목록에 등록된 표현. 클릭하면 해당 셀로 이동
유의어 없음 해당 학생·활동에서 유의어가 발견되지 않음

9-2. 사용 방법

  1. 1 점검 실행 버튼을 클릭하면 전체 기록을 스캔합니다.
  2. 2 결과 목록에서 하이라이트된 표현을 확인합니다.
  3. 3 해당 행을 클릭하면 생기부 작성 탭의 그 셀로 바로 이동합니다. 표현을 수정한 후 다시 점검을 실행합니다.

9-3. 유의어 목록 관리

기본 제공되는 유의어 목록 외에, 설정(Settings) 탭에서 추가 어휘를 등록하거나 수정·삭제할 수 있습니다.

💡

학교·학년별로 다른 제한 표현이 있다면 설정에서 직접 추가하세요. 추가된 어휘는 DB 파일에 저장되므로 해당 파일을 열 때마다 유지됩니다.
