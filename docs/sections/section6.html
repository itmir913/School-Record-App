6. 생기부 작성

좌측 메뉴에서 생기부 작성(Write) 탭을 선택합니다. 영역과 활동을 미리 설정해야 이 화면을 사용할 수 있습니다.

6-1. 화면 구성

실제 화면 — 생기부 작성(Write)

영역
학년 반 번호 이름 합계 학급자치회
1 1 1 학생A 82 Bytes
학급 회의에서 적극적으로 의견을 제시하였으며 친구들의 이야기를 경청하고 조율하는 리더십을 발휘함.
1 1 2 학생B 118 Bytes
학급 대표로 다양한 활동을 기획하고 주도하여 학급의 단결력 향상에 기여하였으며, 조용하지만 신뢰를 주는 리더십을 보여줌.
1 1 3 학생C 1040 Bytes
전략적 사고와 뛰어난 통솔력으로 학급 활동 전반을 이끌었으며, 갈등 상황에서도 냉정하고 합리적인 판단으로 구성원 모두의 신뢰를 받음. 향후 더욱 성장할 것으로 기대되며 다양한 역량을 발휘할 리더임이 분명함. 앞으로도 꾸준히 성장하길 바람.
1 2 2 학생E 0 Bytes
화면 요소 설명
영역 선택 상단 드롭다운에서 영역을 선택하면 생기부 작성 화면이 나타남
글자크기 변경 에디터의 글자 크기를 조절할 수 있습니다.
틀고정 ON 스크롤 시 상단 헤더 행을 고정하거나 해제합니다.
스마트스크롤 ON 학생 이름 영역에서 스크롤하면 상하 스크롤, 셀에서 스크롤하면 좌우 스크롤로 동작합니다.
셀높이 고정 모든 셀을 동일한 고정 높이로 표시할지, 내용에 따라 가변 높이로 표시할지 선택합니다.
빈 학생 ON 아무 내용도 작성되지 않은 학생을 노란색으로 강조 표시합니다.
바이트수 초과 학생 영역의 바이트 한도를 초과한 학생은 붉은색으로 강조 표시됩니다.
History 해당 활동의 과거 생기부 기록을 누적 저장하여 버전을 관리합니다.

6-1-1. 활동 합산 → 최종 문장 완성

같은 영역에 속한 활동들의 내용은 이어 붙여져 해당 영역의 최종 문장이 됩니다. 내보내기(Export) 시 자동으로 합산되므로, 각 활동별로 독립적으로 내용을 작성하면 됩니다.

예시 — 영역 "1반 자율활동" / 학생A

활동①
학급자치회
학급 회의에서 적극적으로 의견을 제시하였으며 친구들의 이야기를 경청하고 조율하는 리더십을 발휘함.
활동②
스포츠데이
팀 경기에서 구성원들의 사기를 북돋우며 협동심을 발휘하였고, 경기 후 반성 시간에 자신의 역할을 돌아보는 성숙한 태도를 보임.

6-2. 내용 입력 및 자동 저장

  • • 그리드의 셀을 클릭하면 입력 모드가 됩니다. 바로 타이핑을 시작할 수 있습니다.
  • • 입력이 멈추면 자동으로 저장됩니다. 별도 저장 버튼은 없습니다.

6-3. 셀 히스토리

각 셀은 변경 이력이 누적 저장됩니다. 셀 하단 History를 클릭하면 이전 버전을 확인하거나 현재 문장을 저장할 수 있습니다.
