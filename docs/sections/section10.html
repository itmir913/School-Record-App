10. 내보내기 (Export)

좌측 메뉴에서 내보내기(Export) 탭을 선택합니다. 작성이 완료된 생기부를 NEIS 업로드에 사용할 엑셀 파일로 추출합니다. 추출된 파일을 NEIS에 업로드하면 생기부 입력이 완료됩니다.

내보내기 절차

  1. 1 내보낼 영역을 선택합니다. 영역별로 별도 파일이 생성됩니다.
  2. 2 내보낼 학생 범위를 선택합니다. 전체 또는 특정 학생만 선택하여 추출할 수 있습니다.
  3. 3 요약 정보(총 학생 수, 바이트 초과 학생 수)를 확인합니다. 바이트 초과 학생이 있을 경우 경고가 표시됩니다.
  4. 4 면책 안내문을 확인합니다. 내보낸 파일을 NEIS에 입력하기 전 반드시 교사가 내용을 최종 검토해야 합니다.
  5. 5 엑셀 내보내기 버튼을 클릭하고 저장 위치를 지정합니다.
⚠️

바이트 초과 학생이 있는 상태로 내보내기를 진행하면 NEIS 업로드 시 오류가 발생할 수 있습니다. 내보내기 전 생기부 작성 화면에서 모든 학생의 바이트 수를 확인하세요.

출력 파일 구조

내보내기로 생성된 엑셀 파일은 NEIS 일괄 업로드 형식에 맞춰져 있습니다.
