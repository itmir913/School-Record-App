1. 시작하기 전에

1-1. 시스템 요구 환경

  • 🖥️운영체제: Windows 10 / 11 (x64) 또는 macOS 12 이상
  • 💾저장 공간: 최소 100MB 이상의 여유 공간 권장
  • 🔒네트워크: 불필요 (완전 오프라인 동작)

1-2. 설치 및 첫 실행

🌐

윈도우 설치버전·무설치버전·macOS 버전 모두 공식 페이지에서 다운로드할 수 있습니다.
https://luminousky.com/teacher-utility-kit/neis-editor/

Windows — 설치형 (.exe)

  1. 1 공식 페이지에서 windows-x64-setup.zip 파일을 다운로드합니다.
  2. 2 압축을 해제한 뒤 neis-editor_x.x.x_x64-setup.exe를 실행하고 설치를 완료합니다.
  3. 3 설치 완료 후 바탕화면 또는 시작 메뉴의 neis-editor 바로가기로 실행합니다.

Windows — 무설치형 (Portable)

  1. 1 windows-x64-portable.zip을 다운로드하여 원하는 폴더에 압축 해제합니다.
  2. 2 폴더 안의 neis-editor.exe를 바로 실행합니다. 별도 설치 과정이 없습니다.
⚠️

Windows에서 "Windows의 PC 보호" 경고가 나타날 수 있습니다. 추가 정보 → 실행을 클릭하면 정상적으로 실행됩니다. 이 프로그램은 어떠한 데이터도 외부로 전송하지 않습니다.

1-3. DB 파일 생성 및 열기

프로그램을 실행하면 홈 화면이 나타납니다. 아래 두 버튼으로 작업을 시작합니다.

실제 화면 — 홈

All-in-One 학교생활기록부 에디터

학생부를 체계적으로 작성하기 위한 교육용 프로그램

새 학생부 만들기
기존 파일 열기
신규

새 학생부 만들기

파일 저장 위치와 이름을 지정하면 새 .db 파일이 생성되고 작업 화면으로 이동합니다.

기존

기존 파일 열기

이전에 작업한 .db 파일을 선택합니다. 암호화된 파일은 비밀번호 입력창이 나타납니다.

1-4. 암호화 설정

DB 파일에 비밀번호를 설정하면 파일을 열 때마다 비밀번호를 입력해야 합니다. 설정은 좌측 메뉴 하단의 설정(Settings) 탭에서 변경합니다.

🔑

비밀번호를 분실하면 해당 DB 파일을 복구할 수 없습니다. 암호화를 사용하는 경우 반드시 안전한 곳에 비밀번호를 별도 보관하세요.
