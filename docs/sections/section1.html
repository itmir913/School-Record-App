1. 시작하기 전에

1-1. 시스템 요구 환경

  • 🖥️운영체제: Windows 10 / 11 (x64) 또는 macOS 12 이상
  • 💾저장 공간: 최소 100MB 이상의 여유 공간 권장
  • 🔒네트워크: 불필요 (완전 오프라인 동작)

1-2. 설치 및 첫 실행

Windows — 설치형 (.exe)

  1. 1 공식 GitHub 릴리즈 페이지에서 windows-x64-setup.zip 파일을 다운로드합니다.
  2. 2 압축을 해제한 뒤 neis-editor_x.x.x_x64-setup.exe를 실행하고 설치를 완료합니다.
  3. 3 설치 완료 후 바탕화면 또는 시작 메뉴의 neis-editor 바로가기로 실행합니다.

Windows — 무설치형 (Portable)

  1. 1 windows-x64-portable.zip을 다운로드하여 원하는 폴더에 압축 해제합니다.
  2. 2 폴더 안의 neis-editor.exe를 바로 실행합니다. 별도 설치 과정이 없습니다.

macOS — 설치형 (.dmg)

  1. 1 macos-setup.dmg를 다운로드하여 열고, 앱을 Applications 폴더로 드래그합니다.
  2. 2 처음 실행 시 macOS Gatekeeper 경고가 나타날 수 있습니다. 시스템 설정 → 개인 정보 보호 및 보안에서 "그래도 열기"를 클릭하세요.
⚠️

Windows에서 "Windows의 PC 보호" 경고가 나타날 수 있습니다. 추가 정보 → 실행을 클릭하면 정상적으로 실행됩니다. 이 프로그램은 어떤 데이터도 외부로 전송하지 않습니다.

1-3. DB 파일 생성 및 열기

이 프로그램은 모든 데이터를 하나의 .db 파일에 저장합니다. 프로그램을 실행하면 홈 화면이 나타나며, 아래 두 가지 방법으로 작업을 시작할 수 있습니다.

신규

새 DB 파일 만들기

홈 화면에서 "새 파일 만들기" 버튼을 클릭하고 저장 위치와 파일명을 지정합니다. 생성 즉시 작업 화면으로 이동합니다.

기존

기존 DB 파일 열기

홈 화면에서 "파일 열기" 버튼을 클릭하거나, 최근 파일 목록에서 이전에 작업한 파일을 선택합니다.

1-4. 암호화 설정

DB 파일에 비밀번호를 설정하면 파일을 열 때마다 비밀번호를 입력해야 합니다. 학생 개인정보 보호가 필요한 경우 활성화하세요. 설정은 작업 화면 좌측 메뉴 하단의 설정(Settings) 탭에서 변경할 수 있습니다.

🔑

비밀번호를 분실하면 해당 DB 파일을 복구할 수 없습니다. 암호화를 활성화하는 경우 반드시 안전한 곳에 비밀번호를 기록해 두세요.
