5. 활동 관리

좌측 메뉴에서 활동(Activity) 탭을 선택합니다. 활동은 영역 안의 세부 기록 단위입니다. 각 영역에 활동을 하나 이상 만들어야 생기부를 작성할 수 있습니다.

5-1. 활동 추가

실제 화면 — 활동(Activity) 화면 상단

활동(Activity) 관리

각 영역의 세부 활동을 등록합니다.

활동 추가
1 우측 상단 활동 추가 클릭 → 팝업에서 소속 영역과 활동 이름을 입력하고 저장합니다.
2 활동이 연결될 영역을 드롭다운에서 선택합니다. 영역이 없으면 먼저 4장 영역 구성을 진행하세요.

5-2. 활동 카드 구성 요소

추가된 활동은 영역별로 그룹화되어 카드 형태로 나타납니다.

실제 화면 — 활동 카드 (영역 "1반 자율활동")

1반 자율활동
1반 자율활동

학급자치회

기록 완료 3 / 3

1반 자율활동

스포츠데이

기록 완료 1 / 3

1반 자율활동

수학여행

기록 완료 0 / 3

요소 설명
영역명 파란색: 배정 학생 전원 기록 완료
영역명 노란색: 일부 학생 기록 미완료
영역명 빨간색: 기록 없음
기록 완료 N / M 전체 M명 중 N명이 이 활동의 내용을 작성 완료한 수

5-3. 활동 순서 변경

활동 카드를 드래그하여 이동하면 같은 영역 안에서 순서를 바꿀 수 있습니다. 최종 내보내기 엑셀의 셀 내용도 이 순서대로 활동이 합산됩니다.
