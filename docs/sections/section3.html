3. 학생 관리

좌측 메뉴에서 학생(Students) 탭을 선택합니다. 학생은 학년·반·번호·이름의 네 가지 정보로 관리됩니다.

3-1. 학생 개별 추가

  1. 1 화면 오른쪽 위의 + 학생 추가 버튼을 클릭합니다.
  2. 2 팝업 창에서 학년, 반, 번호, 이름을 입력하고 저장합니다.

3-2. 엑셀 일괄 추가

기존에 사용하던 출석부(명렬표) 엑셀 파일을 그대로 불러올 수 있습니다.

  1. 1 화면 오른쪽 위의 일괄 추가 버튼을 클릭합니다.
  2. 2 파일 선택 창에서 .xlsx 또는 .csv 파일을 선택하거나 화면에 끌어다 놓습니다.
  3. 3 파일이 인식되면 각 열이 학년·반·번호·이름 중 어느 항목에 해당하는지 매핑 화면이 표시됩니다. 열 이름이 표준 형식이면 자동으로 매핑됩니다.
  4. 4 미리보기로 데이터를 확인한 뒤 가져오기를 클릭합니다.

자동 인식되는 열 이름 예시

항목 인식되는 열 이름
학년 학년, grade
반 반, class, 학급, 반번호
번호 번호, number, 번, 출석번호
이름 이름, name, 성명, 학생명, 학생이름

학생 수정 및 삭제

학생 목록에서 해당 학생의 행을 클릭하면 수정 팝업이 열립니다. 삭제 버튼을 누르면 해당 학생의 모든 생기부 기록도 함께 삭제되므로 주의하세요.
