8. 텍스트 치환 (Replace)

좌측 메뉴에서 텍스트 치환(Replace) 탭을 선택합니다. 특정 단어나 문구를 전체 학생의 생기부에서 한 번에 찾아 바꾸는 기능입니다. 반복 사용된 표현 수정, 맞춤법 오류 일괄 정정, 금지 어구 교체에 유용합니다.

기본 제공 규칙

프로그램에는 자주 쓰이는 치환 규칙이 기본으로 내장되어 있습니다. 줄바꿈 통일, 연속 공백 정리 등의 규칙이 포함되어 있으며, 활성화 여부를 개별로 설정할 수 있습니다.

규칙 설명 찾을 내용 바꿀 내용
연속 줄바꿈 제거 \n+ \n
연속 공백 제거   +  

사용자 정의 규칙 추가

  1. 1 + 규칙 추가 버튼을 클릭합니다.
  2. 2 찾을 내용과 바꿀 내용을 입력합니다. 정규식을 사용하려면 정규식 사용 옵션을 활성화합니다.
  3. 3 규칙을 저장하면 목록에 추가됩니다. 규칙은 나중에 수정하거나 삭제할 수 있습니다.

사용자 정의 규칙 예시

상황 찾을 내용 바꿀 내용
금지어 교체 최우수 탁월한 역량을 보임
오타 수정 참여하였음 참여함

치환 실행

  1. 1 적용할 규칙을 체크하고 치환 실행 버튼을 클릭합니다.
  2. 2 영역과 학생 범위를 선택할 수 있습니다. 전체 또는 특정 영역만 적용 가능합니다.
  3. 3 미리보기로 변경 사항을 확인한 뒤 적용합니다.
⚠️

치환은 즉시 DB에 반영됩니다. 치환 전 스냅샷을 저장해두면 원상복구가 가능합니다.
