4. 영역 구성

좌측 메뉴에서 영역(Area) 탭을 선택합니다. 영역은 NEIS에 입력할 생기부 항목(자율활동, 진로활동, 교과세특 등) 하나에 해당합니다.

4-1. 영역 추가

실제 화면 — 영역(Area) 화면 상단

영역(Area) 관리

자율활동, 진로활동, 동아리활동, 세부능력특기사항 등 생기부 대분류 영역을 설정합니다.

영역 추가
1 우측 상단의 영역 추가 버튼을 클릭합니다.
2 팝업 창에서 영역 이름과 바이트 수 제한을 입력하고 저장합니다.

4-4. 학생 배정

영역 카드 하단의 학생 배정 버튼을 클릭하면 학생을 선택할 수 있는 창이 열립니다.

  • • 반 단위로 전체 선택하거나 개별 학생을 체크박스로 선택합니다.
  • • 엑셀로 일괄 배정 버튼을 누르면, 학생 명단을 불러와서 자동 배정할 수도 있습니다.
  • • 해당 영역에 배정된 학생만 생기부 작성 탭의 그리드에 표시됩니다.
