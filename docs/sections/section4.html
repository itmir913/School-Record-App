4. 영역 구성

좌측 메뉴에서 영역(Area) 탭을 선택합니다. 영역은 생기부의 항목 단위입니다. 자율활동, 진로활동, 동아리활동, 교과세특 각각이 하나의 영역이 됩니다.

4-1. 영역 생성 및 바이트 제한 설정

  1. 1 화면 오른쪽 위의 + 영역 추가 버튼을 클릭합니다.
  2. 2 팝업 창에서 영역 이름을 입력합니다. (예: 1반 자율활동, 프로그래밍 A반)
  3. 3 최대 바이트(Byte) 값을 입력합니다. NEIS 기준에 맞는 값을 입력하세요. 이 값을 초과하는 학생의 셀은 생기부 작성 화면에서 붉은색으로 강조 표시됩니다.

NEIS 바이트 제한 참고 (2025년 기준 예시 — 실제 공문 확인 필수)

생기부 항목 최대 바이트
자율활동 1,500 byte
진로활동 1,500 byte
동아리활동 500 byte
교과세특 (과목당) 학교급·과목별 상이

4-2. 학생 배정

영역을 생성한 후에는 해당 영역에서 생기부를 작성할 학생을 배정해야 합니다.

  1. 1 영역 목록에서 해당 영역의 학생 배정 버튼을 클릭합니다.
  2. 2 팝업 창에서 이 영역에 배정할 학생을 체크합니다. 학년·반 필터로 빠르게 선택할 수 있습니다.
  3. 3 저장을 클릭합니다. 이후 생기부 작성 화면에서 해당 영역을 선택하면 배정된 학생만 표시됩니다.
💡

담임교사라면 학급 전원을 배정하고, 교과교사라면 담당 분반 학생만 선택하세요. 배정은 언제든지 변경할 수 있습니다.

영역 수정 및 삭제

영역 카드에서 수정 아이콘을 클릭하면 이름·바이트 제한을 변경할 수 있습니다. 영역을 삭제하면 해당 영역에 속한 모든 활동과 생기부 기록이 함께 삭제됩니다.
