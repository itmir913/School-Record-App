# 개인정보처리방침 (Privacy Policy)

**All-in-One 학교생활기록부 에디터**(이하 '본 프로그램')는 학교 현장의 선생님들께서 학생들의 민감한 정보를 안전하게 다루실 수 있도록 보안과 프라이버시를 최우선으로 설계되었습니다.

본 프로그램의 개인정보 및 데이터 처리 방침은 다음과 같습니다.

### 1. 외부 서버로의 데이터 수집 및 전송 없음 (100% 오프라인 작동)
본 프로그램은 오프라인 기반으로 작동하는 데스크톱 애플리케이션입니다.
사용자가 입력하는 학생 명렬표, 세부능력 및 특기사항, 창의적 체험활동 등 **어떠한 데이터도 외부 서버로 수집, 전송, 또는 가공되지 않습니다.** 프로그램 내에 자체적인 통계 수집기나 트래커(Tracker) 또한 전혀 포함되어 있지 않습니다.

### 2. 데이터의 로컬(Local) 저장
사용자가 작업한 모든 정보는 **사용자가 프로그램을 설치하고 실행하는 PC 내부에 로컬 데이터베이스(DB) 파일 형태로만 저장**됩니다.

### 3. ⚠️ 사용자의 데이터 보안 및 관리 책임 (중요)
모든 데이터가 사용자의 PC에만 저장되므로, 데이터 유출을 약기 위해 본 프로그램 사용자의 주의가 필요합니다.
- 프로그램이 생성하는 DB 파일에는 **학생들의 매우 민감한 개인정보(이름, 성적, 활동 내역 등)가 평문으로 포함**될 수 있습니다.
- 따라서 해당 DB 파일이 저장된 PC의 보안(비밀번호 설정, 백신 사용 등)을 철저히 유지해 주시기 바랍니다.
- 업무용 PC가 아닌 공용 PC에서의 사용을 지양하고, USB나 클라우드(Google Drive, OneDrive 등)에 DB 파일을 백업할 경우 암호화하여 압축하는 등 각별한 주의를 당부드립니다.
- 추후 DB 파일 자체에 암호화를 설정하는 옵션이 도입된다면, 반드시 DB 파일을 자체 암호화하여 보관하시기 바랍니다.

### 4. 변경 사항 안내
본 개인정보처리방침은 프로그램의 업데이트나 기능 변경에 따라 수정될 수 있으며, 중대한 변경 사항이 있을 경우 GitHub 릴리즈 노트 및 공식 홈페이지를 통해 안내해 드립니다.
